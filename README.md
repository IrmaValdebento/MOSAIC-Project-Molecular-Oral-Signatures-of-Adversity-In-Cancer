# MOSAIC-Project-Molecular-Oral-Signatures-of-Adversity-In-Cancer
MOSAIC Project (LOI 2026). R code for Mendelian Randomization analyses identifying divergent pathways of social adversity in oral cancer. Scripts demonstrate Education as a 'biological shield' (reducing epigenetic age) versus Material Deprivation driving behavioral risk (smoking), validating our novel 'society-to-cells' framework.
